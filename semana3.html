Arreglos con JavaScript

    Los arreglos son heterogéneos porque puede guardar varios tipos de datos. Los arreglos son dinámicos en JavaScript.

  • Creación
  • Acceso
  • Propiedades
  • Iteración
  • Características particulares

    Creación

        var arreglo = new Array();
        var arreglo = [];
        var arreglo = Array('hola', 123, true);
        var arreglo = ['hola', 123, true];

    Acceso

        var x = arreglo[0];
        arreglo[0] = 'mundo';
        arreglo[10] = 3456;

    Matriz

        var m = [[1,2,3],[4,5,6]]
        var m = Array (Array (1,2,3), Arrary (4,5,6));

Sintaxis de la función

  1. Declarar la función usando var y un nombre para la función.
  2. Usar comando function para indicar que se trata de una función.
  3. El fragmento entre paréntesis se llama Parámetro; al parámetro se le va a asignar un valor especifico cuando llamemos a la función.
  4. Escribe el código entre { }

Ejemplos.....

    1) Declaración
        function nombreFuncion (parametro){
              Instrucciones
        }

    2) Expresión
        var nombreFuncion = function [nombre Opcional] (parametros){
              Instrucciones
        }

    Las funciones anónimas no necesitan nombre.

Comando return

El comando return simplemente le devuelve al programador el valor que resulta de una función.

Variables globales vs locales

El ámbito puede ser global o local

  • Las variables definidas fuera de una función son accesibles en cualquier parte una vez que han sido declaradas. Son llamadas variables globales y su ámbito es global
  • Las variables definidas dentro de una función son variables locales. No se puede acceder a ellas fuera de la función.
