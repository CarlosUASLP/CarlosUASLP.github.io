Sintaxis de la funcion

  1. Declarar la funcion usando var y un nombre para la funcion.
  2. Usar comando function para indicar que se trata de una funcion.
  3. El fragmento entre parentesis se llama Parametro; al parametro se le va a asignar un valor especifico cuando llamemos a la funcion.
  4. Escribe el codigo entre { }

1) Declaracion
function nombreFuncion (parametro){
      Instrucciones
}

2) Expresion
var nombreFuncion = function [nombre Opcional] (parametros){
      Instrucciones
}
Las funciones anonimas no necesitan nombre.

Comando return

El comando return simplemente le devuelve al programador el valor que resulta de una funcion.

Variables globales vs locals

El ambito puede ser global o local

  • Las variables definidas fuera de una funcion son accesibles en cualquier parte una vez que han sido declaradas. Son llamadas vaiables globales y su ambito es global
  • Las variables definidas dentro de una funcion son variables locales. No se puede acceder a ellas fuera de la funcion.
