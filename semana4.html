Bucle for

  • Cada bucle for hace uso de una variable de conteo. La primer parte del bucle for le dice a la computadora que comience de un valor de 1 para counter.
  • La segunda parte del bucle es donde podemos controlar el fin del bucle.
  • La tercera parte es la frecuencia con la que se van a realizar las iteraciones.

Ejemplo

    for (var counter = 1; counter < 6; counter++ ) {
        console.log(counter);
    }

Conoce los arreglos.....

    Los arreglos son variables que pueden almacenar números y/o cadenas.

Arreglos

  • Almacenan listas de datos.
  • Pueden almacenar diferentes tipos de datos al mismo tiempo.
  • Están ordenados de forma que la posición de cada dato es fija.

Ejemplo

    var nombres = [´Carlos´, ´Mario´, ´Diana´, ´Paola´];

Acceso a los datos del arreglo

    La posición de los elementos en un arreglo es fija. Asi que solamente necesitamos el nombre del arreglo y la posición de los datos que queremos.

  • La posición o el índice de cada dato comienza a contarse desde 0, no desde 1.
  • Primer elemento arreglo[0];
  • Tercer elemento arreglo[2];
  • Se utiliza la propiedad length del arreglo para poder saber la cantidad de elementos que se encuentran actualmente dentro del arreglo.

Bucle while

    El bucle while es ideal para cuando quieres usar un bucle, pero no sabes cuantas veces tendrás que ejecutarlo.

Sintaxis.....

    while (condición){
        //Realizar acciones.
    }

     Mientras que la condición sea verdadera (TRUE) el código se ejecuta hasta que la condición sea FALSE.

Bucles infinitos.....

    Si le damos una condición TRUE a un bucle while y no creas una forma de que esa condición posiblemente se convierta a FALSE, el bucle se quedara ejecutando para siempre.

Ejemplo

    var bucle = function(cantidad){
         while(cantidad < 4)
        {        
            console.log('Estoy dentro del bucle.');
            cantidad++;
        }
    };

Conclusiones entre los bucles for y while

    Los bucles for son geniales para realizar la misma tarea una y otra vez, cuando sabes con anticipación cuantas veces tendrás que repetir los bucles.

    Los bucles while son ideales cuando tienes que hacer un bucle pero no sabes con anticipación cuantas veces tendrás que ejecutarlo.

Bucles do / while

    A veces quieres estar seguro de que tu bucle se ejecuta al menos una vez, sin importar nada. Si este es el caso, lo que buscas es un bucle do / while.

Sintaxis.....

    var condicionBucle = false;
    do{
            console.log("Voy a dejar de hacer este bucle porque mi condición es: String(condicionBucle)");
    }while(condicionBucle === true)

DOM

Document Object Model

    Representa el documento HTML en forma de una estructura de árbol. El documento se puede manipular fácilmente a través del árbol, es decir, agregar, eliminar, modificar elementos, etc.

  1. Acceder elementos del documento

    document.getElementsByTagName('etiqueta')
    document.getElementByClassName('clase')
    document.getElementById('identificador')

  2. Modificar el contenido de un elemento

    elemento.textContent = 'texto'
    elemento.innerHTML = 'código HTML'

  3. Agregar elementos (nodos) nuevos

    • Crear el elemento (createElement())
    • Crear el contenido (createTextNode())
    • Añadir el contenido al elemento (appendChild())
    • Agregar atributos (setAttribute())
    • Agregar el nuevo elemento al documento (appendChild())
