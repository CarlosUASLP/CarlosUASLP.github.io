Bucle for

  • Cada bucle for hace uso de una variable de conteo. La primer parte del bucle for le dice a la computadora que comienze de un valor de 1 para counter.
  • La segunda parte del bucle es donde podemos controlar el fin del bucle.
  • La tercera parte es la frecuencia con la que se van a realizar las iteraciones.

Ejemplo

for (var counter = 1; counter < 6; counter++ ) {
console.log(counter);
}

Conoce los arreglos.....

Los arreglos son variables que pueden almacenar numeros y/o cadenas.

Arreglos

  • Almacenan listas de datos.
  • Pueden almacenar diferentes tipos de datos al mismo tiempo.
  • Estan ordenados de forma que la posicion de cada dato es fija.

Ejemplo

var nombres = [´Carlos´, ´Mario´, ´Diana´, ´Paola´];

Acceso a los datos del arreglo

La posicion de los elementos en un arreglo es fija. Asi que solamente necesitamos el nombre del arreglo y la posicion de los datos que queremos.

  • La posicion o el indice de cada dato comienza a contarse desde 0, no desde 1.
  • Primer elemento arreglo[0];
  • Tercer elemento arreglo[2];
  • Se utiliza la propiedad length del arreglo para poder saber la cantidad de elementos que se encuentran actualmente dentro del arreglo.

Bucle while

El bucle while es ideal para cuando quieres usar un bucle, pero no sabes cuantas veces tendras que ejecutarlo.

Sintaxis.....

while (condicion){
//Realizar acciones.
}

Mientras que la condicion sea verdadera (TRUE) el codigo se ejecuta hasta que la condicion sea FALSE.

Bucles infinitos.....

Si le damos una condicion TRUE a un bucle while y no creas una forma de que esa condicion posiblemente se convierta a FALSE, el bucle se quedara ejecutando para siempre.

Ejemplo

var bucle = function(cantidad){
while(cantidad < 4)
{
console.log('Estoy dentro del bucle.');
cantidad++;
}
};
bucle(0);

A veces quieres estar seguro de que tu bucle se ejecuta al menos una vez, sin importar nada. Si este es el caso, lo que buscas es un bucle do / while.

Sintaxis.....

do{
console.log("Voy a dejar de hacer este bucle

while(condicionBucle === true) }

Conclusiones entre los bucles for y while

Los bucles for son geniales para realizar la misma tarea una y otra vez, cuando sabes con anticipacion cuantas veces tendras que repetir los bucles.

Los bucles while son ideales cunado tienes que hacer un bucle pero no sabes con anticipacion cunatas veces tendras que ejecutarlo.

DOM

Document Object Model

Representa el documento HTML en forma de una estructura de árbol. El documento se puede manipular facilmente a traves del árbol, es decir, agregar, eliminar, modificar elementos, etc.

  1. Acceder elementos del documento

    document.getElementsByTagName('etiqueta')
    document.getElementByClassName('clase')
    document.getElementById('identificador')

  2. Modificar el contenido de un elemento

    elemento.textContent = 'texto'
    elemento.innerHTML = 'codigo HTML'

  3. Agregar elemntos (nodos) nuevos

    • Crear el elemento (createElement())
    • Crear el contenido (createTextNode())
    • Añadir el contenido al elemento (appendChild())
    • Agregar atributos (setAttribute())
    • Agregar el nuevo elemento al documento (appendChild())
