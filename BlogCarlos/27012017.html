Unidad 2. HTML

Hypertext Markup Languaje

Lenguaje de marcado para hipertexto
  • Las paginas HTML son documentos de texto.
  • HTML usa elementos paa definir la estructura de la pagina.
  • Los elementos se definen por etiquetas encerradas por parentesis angulares < y >.
  • El contenido de un elemento se encierra entre etiquetas.
  • Existen etiquetas de cierre y de apertura
    • <etiqueta> Apertura
    • </etiqueta> Cierre
  • El nombre de la etiqueta de apertura debe ser el mismo que distingue a la etiqueta de cierre con la diagonal.
    • <etiqueta> contenido </etiqueta>
  • Las etiquetas puden tener atributos los cuales se especifican en la etiqueta de inicio.
  • Los atributos contienen informacion adicional acerca del contenido.
    • <etiqueta atributo = "valor" > contenido </etiqueta>
