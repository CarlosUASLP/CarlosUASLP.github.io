Notas de index.html

  • Los comentarios se hacen entre <!-- Comentarios -->
  • DOCTYPE no es una etiqueta, sino una declaracion. Quiere decir que la version de html es la 5.
    <!DOCTYPE html >
  • El elemento <head> contiene informacion para el navegador y no se visualiza.
  • El elemento <title> contiene el nombre de la pagina que se muestra en la pesta√±a del navegador.
  • El elemento <meta> metadatos = datos que describen a los datos. No tiene etiqueta de cierre.
    <meta charset = "utf-8" >
  • El elemento <body> contiene la informacion que se visualizara en el navegador como el contenido de la pagina.
  • Si escribes ! + tabulador se escribe o autocompleta parte de la estructura de HTML.

Notas de estructura.html

  • Dentro del elemento <body> va toda la estructura semantica.
  • La etiqueta <header> es para el buscador, posicionamiento de las paginas.
