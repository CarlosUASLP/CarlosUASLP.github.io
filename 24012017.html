Introducción a la Web

WWW (World Wide Web), es la red de alcance global, creada a principios de los 90´s, gracias a Tim Berners- Lee

Internet es una conexión física entre computadoras. Surge a partir de la segunda guerra mundial (ARPANET)

Tim Berners-Lee también creo:

  • HTTP (Hypertext Transfer Protocol)
  • HTML (Hypertext Markup Languaje)
  • URL (Uniform Resource Locator)

Berners-Lee utilizo la NeXTcube en el CERN y fue el primer servidor web del mundo.
