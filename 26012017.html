Datos importantes sobre la Deep Web

La web que todos conocemos representa solo el 15%, el otro 85% es información secreta, privada y mayormente prohibida

La Deep Web, es la gran parte del contenido de internet que es inaccesible a los motores de búsqueda convencionales

Que puedes encontrar en la deep web:

  • mensajes de correo electrónico
  • mensajes de chat
  • contenido privado en sitios de redes sociales
  • extractos bancarios electrónicos
  • registros de salud electrónicos

El termino deep web fue acuñado por BrightPlanet en un libro del 2001 titulado "The deep web: Surfacing Hidden Value". El contenido de la deep web es ilegal

1 bit coin = 283 dólares. El bit coin es una moneda digital que se usa para hacer la mayoría de las transacciones en la Deep Web y esta fuera del control de los bancos estatales.

El contenido que existe ahí es privado, confidencial y en muchos casos hasta ilegal.

URL diferentes
Las url en la deep web están encriptadas y tienen el siguiente formato: as12fd2d.onion

Acceso limitado
Los navegadores o browsers que usamos como (firefox, chrome, explorer) no están diseñados para permitir el acceso a este tipo de urls.

Tor (The Onion Router)
Es un proyecto cuyo objetivo principal es el desarrollo de una red de comunicaciones distribuido de baja latencia y superpuesta sobe internet, en la que el encaminamiento de los mensajes intercambiados entre los usuarios no revela su identidad, es decir su dirección IP
