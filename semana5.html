¿Para que nos puede servir la libreria jQuery?

  • La interacción con HTML.
  • La manipulación DOM.
  • La manipulación de CSS.
  • Manejo de eventos.
  • Animaciones.
  • Interacciones AJAX.
  • Garantiza la compatibilidad entre navegadores.

Todo se realiza a traves de una función llamada: jQuery() recibe como parametro una expresión CSS y regresa un objeto jQuery .
Sintaxix.....
var objeto = jQuery(selector CSS)

Tipos de selectores

  • * Obtenemos todos los elementos.
  • # para los ID.
  • . (punto) para las clases.

Ejemplo

var todos = jQuery('*')
Nos regresa un arreglo con todos los elementos.

Filtros basicos

  • :first
    var primero = jQuery('*:fisrt')
  • :last
    var ultimo = jQuery('*:last')
  • :even
    posicion par
  • :odd
    posicion impar
  • :eq(index)
    posicion igual que index.
    var segundo = jQuery('* : eq(1)')
  • :gt(index)
    mas alla de la posicion.
  • :lt(index)
    antes de una posicion.
  • :not(selector)
    var sinParrafo = jQuery('* : not(p)')

Metodos de jQuery

Contenido

  • .html()
  • .text()

Elementos

  • .before()
  • .after()
  • .prepend()
  • .append()

Ejemplo:
.before() <etiqueta> .prepend() contenido .append() </etiqueta> .after()

Atributos

  • .css()
  • .attr()
  • .addClass()
  • .removeClass()

Eventos

  • .ready()
       Se ejecuta al momento de terminar de crear el DOM.
  • .load()
       Se ejecuta al momento de termiar de cargar todos los recursos. (imagenes, scripts, archivos de audio, etc.)

Ejemplos

$(document).ready(funcion)
$(document).load(funcion)
Pero tambien lo podemos hacer abreviado $(funcion)
