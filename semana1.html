¿Que son las reglas?

Las reglas sirven para cambiar los elementos de HTML. Las reglas permiten cambiar la apariencia (presentación) de los elementos HTML.

Sintaxis general de CSS

selector { declaraciones }

Declaraciones CSS

propiedad : valor;

Propiedades

  • colores
  • fuentes
  • márgenes
  • etc

Ejemplos

h1 { color : blue; }
p { font-size : 20px; }

Tipos de estilos

  1. Por default (navegador)
    Es el que ya viene integrado en el navegador
  2. Del usuario (archivo local)
  3. Hoja de estilo externa
    Utiliza la etiqueta <link> para hacer referencia al archivo
    Ejemplo:
    <head>
    < link href = "estilos.css" type = "text/css" rel = "stylesheet" >
    </head>
  4. Estilo interno
    Esta adentro del mismo archivo HTML y CSS. Utiliza la etiqueta <style> No es muy recomendable.
    Ejemplo:
    <head>
    <style>
    h1 { color : blue; }
    </style>
    </head>
  5. Estilo en línea
    Se asigna la propiedad y valor como atributo del elemento.
    Ejemplo:
    < h1 style = " color : blue; "> Titulo </h1>

Selectores CSS

1) Selector de tipo
Selecciona un tipo de elemento HTML y aplica el estilo a todos los elementos del mismo tipo.
Ejemplo:
p { text-align : justify; }
2) Selector descendente
Aplica el estilo a los elementos que están contenidos dentro de otros elementos.
Ejemplo:
p b { color : red; }
3) Selector hijo
Similar al descendente pero más restrictivo. Aplica el estilo solamente a los elementos que sean hijos directos de otro elemento.
Ejemplo:
p > b { color : blue; }
4) Selector por ID
Un ID es un atributo de un elemento HTML y se utiliza para aplicar un estilo de manera particular.
Ejemplo:
<p id = "introducción"> Este es el párrafo de la introducción </p>
#introducción{ color : blue; }
5) Selector por clase
Aplica estilo a un conjunto de elementos.
Ejemplo:
<p class = "especial"> Este es un párrafo especial </p>
.especial { color : red; }
6) Selector universal
Aplica el mismo estilo a "todos" los elementos de la página.

Los estilos se heredan, si aplicas un estilo a un elemento, todos los elementos dentro de el tendrán el mismo estilo, pero se puede modificar por medio de declaraciones adicionales en cascada.
